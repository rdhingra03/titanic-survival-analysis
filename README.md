# titanic-survival-analysis
R-based analysis of Titanic survival using statistical and machine learning models, including regression, classification trees, and model comparison.

This project analyzes factors that influenced survival on the Titanic using statistical and machine learning models in R.

## Data
- 622 randomly sampled passengers
- Variables include:
  - Passenger class
  - Gender
  - Fare
  - Family size (SibSp, Parch)
  - Port of embarkation

## Methods
- Exploratory Data Analysis (EDA)
- LDA and QDA classification
- Logistic regression
- Classification trees
- Train/test split for model evaluation

## Key Findings
- Gender is the strongest predictor of survival
- Higher passenger class is associated with higher survival rates
- Classification tree model achieved the lowest overall error (~19.5%)
- Logistic regression performed best for predicting survivors
